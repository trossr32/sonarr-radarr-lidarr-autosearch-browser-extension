Privacy policy

The short version: the extension collects nothing, phones home to no one, and everything stays on your machine.

Policy

This extension does not capture any personal information.

All browser extension settings are saved locally on your machine using the browser's built-in extension storage functionality.

No analytics or tracking scripts of any kind are included in the extension.

Data handling in detail

  • All settings are stored locally using the browser.storage extension API.
  • API keys never leave your machine except when calling your own configured Servarr endpoints.
  • No telemetry, analytics, or third-party tracking of any kind.
  • Optional donation links open external payment pages, and only through a manual user action.

Backups

The optional settings backup is a file you download to your own machine; it is never uploaded anywhere. Note that it contains the API keys of any configured Servarr instances, so store it securely.

This website

arrsearch.app is a static site hosted on GitHub Pages with no analytics or cookies of its own. Store rating badges are loaded from shields.io and contributor avatars from GitHub; those requests are subject to the respective services' policies.

Questions? Open an issue on GitHub.
